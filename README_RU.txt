# Бектестер стратегии будущего бота

Чистая версия с визуалом v11 и строгой логикой под ТЗ.

## Запуск

1. Распакуйте ZIP.
2. Откройте `index.html` в браузере.
3. Нажмите **Запустить**.

Python не нужен.

## Что внутри

- расчёт по минутным свечам `1m`;
- сборка текущей 4H-свечи из `1m`;
- BUY только по формуле из ТЗ;
- SELL только по формуле из ТЗ;
- одновременно только одна позиция;
- BUY при открытой позиции игнорируется;
- SELL без позиции игнорируется;
- журнал сделок;
- кривая капитала;
- вкладка проверки;
- вкладка ручной сверки.

## Как читать экран

Слева — только основные настройки.  
Сверху — главные метрики.  
В центре — график.  
Снизу — сделки и разбор.

## Управление графиком

- колесо мыши — приближение;
- зажать и двигать мышью — сдвиг;
- Shift + drag — выделение участка;
- двойной клик — сброс масштаба;
- наведение на свечу — точные данные.


## Экспорт сделок CSV

Вкладка «Сделки» показывает ограниченное количество строк, чтобы браузер не зависал на больших периодах. Полный журнал всех закрытых сделок выгружается кнопкой **CSV** справа в строке вкладок. В CSV попадают вход, выход, цены, размер, количество, PnL, BUY/SОLL distance и причины входа/выхода.

Логика продажи:

```text
SELL выполняется только если:
1. текущая 4H-свеча зелёная;
2. (High − Current Price) / Current Price × 100 >= trigger %;
3. закрытие позиции после комиссии и проскальзывания даёт PnL > 0.
```

Если первые два условия выполнены, но PnL <= 0, позиция не закрывается. Бектестер удерживает BTC и ждёт следующего момента, когда одновременно выполнится SELL-формула и выход будет в плюс.

Это изменение сделано по новой бизнес-логике: **не продавать BTC себе в убыток, а удерживать актив при просадке**.
