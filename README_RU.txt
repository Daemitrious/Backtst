# Бектестер стратегии будущего бота

Чистая версия с визуалом v11 и строгой логикой под ТЗ.

## Запуск

1. Распакуйте ZIP.
2. Откройте `index.html` в браузере.
3. Нажмите **Запустить**.

Python не нужен.

## Что внутри

- расчёт по минутным свечам `1m`;
- сборка текущей 4H-свечи из `1m`;
- BUY только по формуле из ТЗ;
- SELL только по формуле из ТЗ;
- одновременно только одна позиция;
- BUY при открытой позиции игнорируется;
- SELL без позиции игнорируется;
- журнал сделок;
- кривая капитала;
- вкладка проверки;
- вкладка ручной сверки.

## Как читать экран

Слева — только основные настройки.  
Сверху — главные метрики.  
В центре — график.  
Снизу — сделки и разбор.

## Управление графиком

- колесо мыши — приближение;
- зажать и двигать мышью — сдвиг;
- Shift + drag — выделение участка;
- двойной клик — сброс масштаба;
- наведение на свечу — точные данные.

## Важно

Это не live-бот и не Testnet-бот. Это бектестер логики будущего бота по ТЗ.
