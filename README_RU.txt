# Бектестер стратегии будущего бота

Чистая версия с визуалом v11 и строгой логикой под ТЗ.

## Запуск

1. Распакуйте ZIP.
2. Откройте `index.html` в браузере.
3. Нажмите **Запустить**.

Python не нужен.

## Что внутри

- расчёт по минутным свечам `1m`;
- сборка текущей 4H-свечи из `1m`;
- BUY только по формуле из ТЗ;
- SELL только по формуле из ТЗ;
- одновременно только одна позиция;
- BUY при открытой позиции игнорируется;
- SELL без позиции игнорируется;
- журнал сделок;
- кривая капитала;
- вкладка проверки;
- вкладка ручной сверки.

## Как читать экран

Слева — только основные настройки.  
Сверху — главные метрики.  
В центре — график.  
Снизу — сделки и разбор.

## Управление графиком

- колесо мыши — приближение;
- зажать и двигать мышью — сдвиг;
- Shift + drag — выделение участка;
- двойной клик — сброс масштаба;
- наведение на свечу — точные данные.

## Важно

Это не live-бот и не Testnet-бот. Это бектестер логики будущего бота по ТЗ.


## Обновление v16: максимальный исторический охват с 2017 года

В этой версии расширена загрузка 1m-свечей. Теперь бектестер может пытаться загружать историю начиная с 01.01.2017 и дальше, пачками по 1000 минутных свечей, без прежнего ограничения примерно в 25 дней. Лимит установлен до 7000 API-запросов, чего достаточно примерно для 7 000 000 минутных свечей.

Важно: логика стратегии не изменена. Как и раньше, расчёт идёт строго по ТЗ: 1m-свечи используются как сырьё, из них собирается текущая незакрытая 4H-свеча, BUY/SELL считаются только по формулам ТЗ, одновременно разрешена только одна позиция.

Если биржа не имеет данных за ранний период или API не отдаёт историю, бектестер не должен молча выдавать демо за реальные данные. В таком случае будет показана ошибка или предупреждение, а для полного офлайн-прогона можно загрузить CSV с минутной историей.
